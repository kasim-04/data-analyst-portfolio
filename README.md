# Data Analyst Portfolio

Здесь собраны мои проекты по анализу данных из различных сфер: медицины, логистики, HR и финансов.

## Проекты

| № | Название | Описание | Технологии | Тип |
|---|----------|----------|------------|-----|
| 1 | Анализ медицинских данных | Анализ медицинской базы данных с историей за 10+ лет. Провёл анализ ключевых метрик по пациентам: кратность обращений, распределение по диагнозам, возрастные и гендерные срезы. Подготовил аналитические отчёты. | `PostgreSQL`, `Python`, `Pandas`, `NumPy` | Стажировка в НПЦ им. Соловьёва |
| 2 | [Оптимизация логистики](https://github.com/kasim-04/parc-clust-sys) | Автоматизация распределения грузов между курьерами с учётом ограничений транспортных средств. Разработал инструмент для автоматического распределения с помощью кластеризации по географии — на выходе готовый план со статистикой по каждой группе. | `SQLite`, `Python`, `Pandas`, `NumPy` | Курсовая работа |
| 3 | [Телеграм-бот для анализа зарплат](https://github.com/kasim-04/job-analysis-bot) | Бот, который помогает пользователю узнать диапазон зарплат по интересующей профессии. Проект сочетает элементы парсинга и аналитики данных. | `Python`, `Requests`, `Pandas`, `NumPy` | Pet-проект |
| 4 | Анализ портфеля акций | Анализ инвестиционного портфеля из акций разных отраслей на основе реальных финансовых данных за год, загруженных с биржи. Рассчитал доходность и ковариацию активов. Применил модель Марковица для подбора оптимального соотношения активов с максимальной доходностью при минимальном риске. | `Excel` | Pet-проект |
